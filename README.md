适用于Flutter的微信SDK，方便快捷。


## 写在前面

 使用Fluwx之前，强烈建议先阅读[微信SDK官方文档](https://open.weixin.qq.com/cgi-bin/showdocument?action=dir_list&t=resource/res_list&verify=1)，
 这有助于你使用Fluwx。Fluwx的api字段名称基本和官方的字段名称是一致的。
